# 小动物单词超市

这是一个不需要安装依赖的纯前端原型，已经包含：

- 首页
- 游戏页
- 5 轮答题逻辑
- 4 选 1 判定
- 结算页
- 所有贴图预留位置

## 怎么打开

直接用 VS Code 打开这个文件夹，然后双击 `index.html`，或者用 Live Server 打开。

## 你只需要贴图的目录

```text
animal-word-mart/
  assets/
    home/
      home-hero.png
    animals/
      rabbit.png
      bear.png
      cat.png
      monkey.png
      panda.png
    foods/
      apple.png
      banana.png
      milk.png
      bread.png
      egg.png
      cake.png
      fish.png
      rice.png
      juice.png
      cookie.png
      carrot.png
      orange.png
```

如果没有图，页面会显示占位框，游戏照样能跑。
